# CLAH Image Analysis

A comprehensive suite of Python programs for 2-photon calcium imaging analysis, with support for 1-photon imaging in progress.

## Overview

CLAH Image Analysis provides tools for:

- 2-photon imaging analysis
- Motion correction and segmentation
- Cell registration
- Unit analysis
- Calcium imaging processing
